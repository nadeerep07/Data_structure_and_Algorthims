# Data Structures and Algorithms

This repository contains implementations and practice problems related to **Data Structures and Algorithms (DSA)**.
It is created as a personal learning resource to improve problem-solving skills and strengthen algorithmic thinking for technical interviews and competitive programming.

---

## Topics Covered

### Data Structures

* Arrays
* Linked Lists
* Stacks
* Queues
* Trees
* Binary Search Trees
* Graphs
* Hash Tables

### Algorithms

* Searching Algorithms
* Sorting Algorithms
* Recursion
* Backtracking
* Dynamic Programming
* Greedy Algorithms
* Graph Traversal (BFS & DFS)

---

## Purpose of the Repository

The goal of this repository is to:

* Practice common DSA problems
* Understand algorithmic concepts
* Improve coding and logical thinking
* Prepare for technical interviews
* Build strong problem-solving skills
